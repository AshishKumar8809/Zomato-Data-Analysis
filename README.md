# 🍽️ Zomato Data Analysis Project  

## 📊 Overview  
This project analyzes Zomato data using Microsoft Excel to find insights about restaurants, ratings, and pricing.  

---

## 🎯 Objectives  
- Analyze restaurant ratings  
- Identify popular cuisines  
- Study price trends  
- Understand location impact  

---

## 🛠️ Tools Used  
- Microsoft Excel  

---

## 📁 Project Files  
- Excel dataset and analysis  
- Presentation (PPT)  
- Project report (DOCX)
